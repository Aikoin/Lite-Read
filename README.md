# Lite Read - 纯净阅读助手

**作者**: Aikoin  
**版本**: 1.0.1

## 简介 (Introduction)
Lite Read 是一款专为 ADHD（注意力缺陷多动障碍）人群及希望获得更佳阅读体验的用户设计的 Chrome 浏览器插件。它能帮助你一键优化网页排版，让阅读变得轻松、专注、无压力。

## 主要功能 (Features)

*   **📏 智能字号调整**：
    *   **相对增减**：在网页原有字号基础上进行微调（如 +2px），完美保留标题与正文的大小层级，不会破坏页面结构。
*   **📐 舒适行间距**：
    *   **动态行高**：支持以像素为单位（如 +5px）增加行高，防止文字拥挤，让视线更顺畅地移动。
*   **🔠 呼吸感字间距**：
    *   **微调字距**：增加汉字之间的空隙，减少“文字墙”带来的压迫感。
*   **🌗 盘古之白 (Pangu Spacing)**：
    *   **自动空格**：一键在中文与英文、数字之间添加空格（如“Hello世界” -> “Hello 世界”），提升排版美感与可读性。
*   **🧹 纯净模式**：
    *   **无痕还原**：所有滑块归零时，插件会自动撤销所有样式注入，确保网页 100% 原样显示，无任何残留。
*   **🧠 智能记忆**：
    *   **状态保持**：在标签页未关闭的情况下，插件会记住你对该页面的调整偏好；再次选中该标签页时，会自动回显当前状态。
    *   **SPA 支持**：完美支持知乎等动态加载内容的网站。

## 安装说明 (Installation)

由于本插件暂未上架 Chrome 商店，你可以通过以下两种方式安装：

### 方式一：直接下载安装包（推荐小白用户 ✅）

1.  **下载**：前往 GitHub 仓库的 **[Releases 页面](https://github.com/Aikoin/Lite-Read/releases)**，下载最新版本的 `lite_read_package.zip` 压缩包。
2.  **解压**：将下载的压缩包解压到一个你容易找到的文件夹（例如桌面）。
3.  **打开扩展管理**：在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
4.  **开启开发者模式**：点击页面右上角的 **“开发者模式” (Developer mode)** 开关将其打开。
5.  **加载插件**：点击左上角的 **“加载已解压的扩展程序” (Load unpacked)** 按钮。
6.  **选择文件夹**：选择你刚才解压出来的文件夹。
7.  **完成**：Lite Read 图标出现在浏览器右上角，即可开始使用！

### 方式二：通过 Git 克隆（适合开发者 💻）

如果你熟悉 Git，可以直接克隆仓库代码使用：

1.  **克隆仓库**：
    ```bash
    git clone https://github.com/Aikoin/Lite-Read.git
    ```
2.  **加载插件**：
    *   打开 Chrome 扩展管理页 `chrome://extensions/`
    *   开启“开发者模式”
    *   点击“加载已解压的扩展程序”
    *   选择克隆下来的 `Lite-Read` 文件夹即可。

## 使用指南 (Usage)

1.  **打开插件**：点击浏览器右上角的 Lite Read 图标。
2.  **调整滑块**：
    *   **字号**：向右拖动放大文字。
    *   **行间距**：向右拖动增加行高。
    *   **字间距**：向右拖动增加字距。
3.  **盘古之白**：点击开关，瞬间优化中英文排版。
4.  **重置**：点击“重置”按钮，一键恢复网页原貌。
5.  **保存预设**：点击“保存预设”可以将当前参数存为全局偏好，在其他网页点击“应用预设”即可快速生效。
